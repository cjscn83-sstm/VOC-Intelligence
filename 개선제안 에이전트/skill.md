# Skill: VOC 개선 제안 (실행 과제 설계·우선순위화·로드맵)

진단을 "무엇을 할 것인가"로 전환하는 스킬.

## 언제 사용하나
- 분석 결과를 담당·기한·지표가 붙은 실행 과제로 바꿔야 할 때
- 단계별 개선 로드맵과 조직별 적용안을 설계할 때

## 절차
1. **레버 매핑** — 현황분석 근거별로 개선 레버 도출
2. **우선순위화** — 임팩트 × 실행 난이도로 정렬
3. **과제화** — 각 과제에 담당 조직·기한·KPI 부여 (측정 가능하게)
4. **로드맵 배치** — 즉시 / 단기 / 중기 / 중장기로 시퀀싱
5. **조직별 적용** — 지사·팀 단위 우선 적용안 작성

## 5대 현장 실행 과제
1. 개통·계약 전 '3대 고지' 서명 체크리스트
2. 고위험 VOC 24시간 에스컬레이션 SLA
3. 취약고객(고령·장애) 판매 프로토콜
4. 출시·성수기 캘린더 연동 선제 대응
5. 캐시백·정산 서면 표준화 + 철회 기준 정렬

## 우선순위 근거
- 근본 원인은 불완전판매(불친절이 아님)
- 격화의 진범은 사후 대응 실패 → 24시간 SLA
- 제도 변화 대응 (단통법 폐지 + 취약고객)
- 예측 가능한 피크는 선제 대응 (출시·성수기 캘린더)

## 출력 형식
- 5대 과제 + 실행 디테일 + 조직별 우선 적용 + 단계별 로드맵
