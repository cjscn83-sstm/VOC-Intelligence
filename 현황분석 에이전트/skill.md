# Skill: VOC 현황 분석 (분류·집계·리스크 스코어링)

내부 VOC 데이터로 "지금 무슨 일이 일어나는가"를 규명하는 스킬.

## 언제 사용하나
- 원본 VOC 엑셀을 정량·정성 분석해 현황을 진단할 때
- 고위험 사례와 조직/시계열 패턴을 데이터로 뽑아야 할 때

## 절차
1. **로딩·검증** — xlsx 로딩, 결측·중복·인코딩 정합성 확인
2. **분류(라벨링)** — 규칙/키워드 + LLM 보조로 카테고리 지정 (불완전판매, 끼워팔기, 설치 품질, 캐시백/정산 등)
3. **스코어링** — 심각도(경/중/고), 감정 격화 스코어 부여 → 고위험(예: 86건) 식별
4. **집계** — 유형 × 지사 × 월 피벗, 이상치·편중 탐지 (지사별 고위험률 편차 등)
5. **사례 추출** — 대표 고위험 사례 상위 5건 심층 리뷰
6. **한계 명시** — 표본·기입 누락·분류 모호성 등 해석 유의점 기록

## 분석 축
- 유형별 (카테고리·건수·핵심 패턴)
- 감정·리스크 (고위험 식별)
- 대표 사례
- 시계열·조직 (월별 추이·지사별 현황)
- 데이터 한계
