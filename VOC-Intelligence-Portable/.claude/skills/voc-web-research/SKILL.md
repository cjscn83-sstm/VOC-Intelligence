# Skill: VOC 웹 리서치 (외부 소스 수집·검증·시사점)

## 언제 사용하나
- 외부 벤치마크(방법론)나 규제·통계 근거가 필요할 때
- 내부 리스크를 제도 변화와 연결해 설명해야 할 때

## 절차
1. **질문 정의** — 방법론 / 통계 / 제도 3영역으로 조사 질문 구성
2. **수집** — WebSearch 1차 소스 → WebFetch 원문 확인·인용
3. **검증** — 발행 기관·최신성·신뢰도 확인, 상호 교차 확인
4. **연결** — 외부 사실을 내부 VOC 패턴과 매핑해 시사점 도출
5. **출처화** — 인용 링크 포함 구조화 의견 산출

## 조사 영역
- VOC 관리 방법론 (폐쇄루프, NPS 이너/아우터 루프)
- 소비자 불만 통계·분쟁 유형
- 제도 환경 (단통법 폐지, 취약고객 보호)
- 당사 VOC 시사점

## 주요 출처
- NPS: The Inner Loop | Bain & Company
- NPS: The Outer Loop | Bain & Company
- 소비자 보호 제도·통계 공개 자료

## 출력 형식
외부 환경 경고 + 방법론/제도 요약 + 출처 목록 + 시사점 (링크 포함)
